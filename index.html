Pulsewaves

An IDL implementation of PulseWaves

Download this project as a .zip file Download this project as a tar.gz file

This project is an IDL implementation of PulseWaves version is 0.3 (revision 9) created by Antoine Cottin.

Carbomap has a joint expertise of more than ten years in lidar full waveform data processing (bathymetric and topographic lidar).

Carbomap takes a step forward, and fully support PulseWaves file format and integrates its support to all of its workflows.

As we want PulseWaves file format to be broadly accepted by the users community, we are releasing an open source IDL library to read and write PulseWaves file. We hope preliminary work will be the ground base to a wide users community effort to develop more tools that will profite to everyone. So don't hesitate to clone and/or fork this project. We want and need you to be involve.

PulseWaves is a Vendor neutral full waveform data format created by Martin Isenburg (RapidLasso) that is retro-compatible with LAS file. The C++ source code and format specification can be found on GitHub at: https://github.com/PulseWaves

For convinency to any IDL developer and for testing, we are hosting the same demo data and the file format specifications available on Martin's github page.

Install

To install simply import the project into your current IDL workspace. Build the project, and you're ready to go

For more information on how to use the class object, please refer to the wiki pages on the github project page
